## Animation Specifications

### 1. Drop Zone Interaction
- **Drag Enter**: Border color transition from gray to blue (200ms ease)
- **Drag Over**: Subtle scale transform (1.02x) + shadow
- **Drop**: Brief flash animation + fade in uploaded image

### 2. Progress Bar
- **Fill**: Smooth width transition (300ms linear)
- **Indeterminate**: Shimmer effect moving left-to-right (1.5s infinite)

### 3. Image Load
- **Fade In**: Opacity 0 â†’ 1 (400ms ease-out)
- **Skeleton**: Animated gradient while loading

### 4. Button Hover
- **Background**: Color darken (150ms ease)
- **Scale**: Slight grow (1.05x) on hover
- **Active**: Pressed state scale (0.95x)

### 5. Queue Item Updates
- **Add**: Slide down + fade in (300ms)
- **Complete**: Check mark bounce animation
- **Remove**: Slide up + fade out (250ms)

## Accessibility Notes

### Keyboard Shortcuts
- `Tab`: Navigate between elements
- `Enter/Space`: Activate buttons
- `Esc`: Cancel operation / close modals
- `Ctrl+V`: Paste image from clipboard (bonus feature)

### Screen Reader Announcements
- "Drop zone, click to upload image"
- "Processing, 67% complete, estimated 5 seconds remaining"
- "Image upscaled successfully, download button available"
- "Error: File size too large"
